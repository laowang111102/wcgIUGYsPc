# 学生宿舍管理系统 python479

## 项目概述

学生宿舍管理系统是一个用于管理学生宿舍信息、分配、保修以及通知等功能的平台。系统旨在提高宿舍管理的效率，简化相关流程，并确保稳定可靠地运行。

## 技术栈

- **开发语言：** Python
- **后端框架：** Flask
- **前端技术：** HTML、CSS、JavaScript、jQuery
- **数据库：** MySQL
- **开发工具：** PyCharm、Navicat

## 运行环境

- **Python版本：** 3.0以上
- **数据库版本：** MySQL 8.0

## 功能模块

- **宿舍分配：** 管理员可以对学生宿舍进行分配和管理。
- **宿舍保修：** 学生可以通过系统提交宿舍维修请求。
- **宿舍查询：** 学生和管理员可以查询宿舍信息。
- **学生信息管理：** 管理学生基本信息。
- **宿舍信息管理：** 管理宿舍楼信息、房间信息等。
- **通知管理：** 发布和查看宿舍相关通知。

## 系统角色

- **学生：** 提交保修请求、查询宿舍信息和通知。
- **管理员：** 管理宿舍分配、宿舍信息、学生信息以及发布通知。
